# BDD Mantine App

A modern React application built with **Behavior-Driven Development (BDD)** testing methodology and **AI-powered development**, using Next.js 15 and Mantine UI framework.

> 🚀 **AI-Generated Project**: This entire project, including all code, tests, and documentation, was generated using Large Language Models (LLMs) following BDD principles. It demonstrates the power of combining AI-assisted development with rigorous testing methodologies.

## 🎯 Project Goals

This project demonstrates how to combine **BDD testing methodology with AI-powered development** to build a video management system with comprehensive test coverage. The project emphasizes:

- **BDD Test-Driven Development**: Using Gherkin syntax to describe feature requirements
- **AI-Assisted Development**: Leveraging LLMs for code generation and testing
- **Modern UI Framework**: Integrating Mantine 8 for beautiful user interfaces
- **Complete Test Coverage**: Including unit tests, integration tests, and E2E tests
- **TypeScript Support**: Providing complete type safety

## 🚀 Quick Start

### Requirements

- Node.js 18+
- npm or yarn

### Install Dependencies

```bash
npm install
```

### Development Mode

```bash
npm run dev
```

The application will start at `http://localhost:3000`

### Run Tests

```bash
# Run all tests
npm test

# Watch mode
npm run test:watch

# Run specific tests
npm test -- features/login.steps.tsx
```

### Build for Production

```bash
npm run build
npm start
```
